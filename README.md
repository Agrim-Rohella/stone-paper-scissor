# stone-paper-scissor
Description:
A console-based Stone Paper Scissors game written in Python using loops and the random module. The game is designed as a best of five rounds where the player competes against the computer which is driven by the random module. The program keeps track of user input, score, and announces the winner after 5 rounds or when a player reaches 3 wins.
In order for the game to be played user has to follow the steps given on the console.

Code Details
Uses a while loop to run multiple rounds until a player reaches 3 wins or 5 rounds expire.
